# Skill: Hilfe

## Antwortformat

Markdown mit drei Sektionen:
1. **Abfragen** (Lese-Anfragen)
2. **Schreiben** (Vorschläge mit Confirm)
3. **Wie sprechen?**

Keine Tools nötig — du gibst nur eine fertige Aufzählung aus. Lass die Beispiele konkret aber kurz.

## Beispiel-Antwort (Markdown direkt, NICHT in ``` einschließen!)

**Was ich tun kann:**

**Abfragen:**
- Wie viele Kunden / Rechnungen / Projekte / Ausgaben haben wir?
- Wer schuldet mir noch was?
- Top 5 Kunden 2026
- Liste alle Rechnungen von <Jahr>
- Übersicht / Monatsstand

**Schreiben (du bestätigst im Chat):**
- Buche heute 2,5 h auf <Projekt> — <Beschreibung>
- Schreib eine Rechnung an <Kunde>, 3h Setup à 90 €
- Trag eine Ausgabe ein: 89 € netto Software Adobe
- Leg einen neuen Kunden an: <Name> + Adresse

**Wie sprechen?**
Sprich mit mir wie mit einem Mitarbeiter — du-Form, knapp, konkret. Bei Unklarheit frag ich nach.
