Wie stelle ich ein Backup wieder her?
  1. Container stoppen: docker stop oa
  2. Snapshot herunterladen oder direkt im DATA_DIR finden: {{ data?.backupDir }}/oa-YYYY-MM-DDTHHmmss.sqlite
  3. SQLite-File über die Live-DB kopieren: cp <snapshot> $DATA_DIR/db/accountant.sqlite
  4. Container wieder starten: docker start oa

Die Snapshots sind echte, vollständige SQLite-Dateien — kein Diff, kein Konzept-Hokuspokus. Du kannst sie mit jedem SQLite-Client öffnen (DB Browser for SQLite, DBeaver, sqlite3 CLI).
